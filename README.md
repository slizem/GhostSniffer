# GhostSniffer

Легковесный анализатор сетевых пакетов.

## Возможности
- Захват и анализ TCP/UDP трафика
- Фильтрация по домену

## Зависимости
- Npcap SDK: https://npcap.com/#download

## Сборка
1. Установить Npcap SDK в `C:\npcap-sdk`
2. Открыть `GhostSniffer.sln` в Visual Studio 2019+
3. Собрать в Release x64
4. Запускать от Администратора

## Управление
- СТРЕЛКИ - выбор режима
- ENTER - подтвердить
- ESC - назад в меню
- ПРОБЕЛ - пауза
- TAB - компактный/подробный режим
